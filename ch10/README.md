# Chapter 10 -- 虛擬記憶體 (Virtual Memory) #

## Section Goals ##

* 定義虛擬記憶體，並描述其好處
* 說明如何使用需求分頁將頁面載入記憶體中
* 應用先進先出、最佳和 LRU 頁面替換演算法
* 描述一行程的工作集，並說明其程式局部性的關係
* 描述 Linux、Windows 10 和 Solaris 如何管理虛擬記憶體
* 使用 C 程式語言設計虛擬記憶體管理模擬
