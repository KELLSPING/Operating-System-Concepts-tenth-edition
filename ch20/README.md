# Chapter 20 -- Linux 系統 (The Linux System) #

## Section Goals ##

* 探討 UNIX 作業系統的歷史，從而導出 Linux ，而 Linux 的設計也是根據此原理

* 檢視 Linux 的記憶體管理
* 探討 Linux 如何製作檔案系統和管理 I/O 裝置
